# Meridian 59 Tools
A collection of useful tools for Meridian 59.
## bgf2png
This tool converts a .bgf file into a set of pngs and a json file.
## roo2obj
This tool converts a .roo file into a 3d mesh, outputing a .obj and .mtl file.
